# Blog Reader
## Installation

1. Clone the repository:
  ```sh
  git clone https://github.com/yourusername/blog-reader.git
  cd blog-reader
  ```

2. Install dependencies:
  ```sh
  npm run install
  ```

## Usage

1. Start the application:
  ```sh
  npm run start
  ```

2. Open your browser and navigate to `http://localhost:5173`.

## Running Tests

1. Run the tests:
  ```sh
  npm run test
  ```
